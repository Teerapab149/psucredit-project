# PSU Credit Checker System
> A specialized credit checking system designed exclusively for Prince of Songkla University (PSU) students.

## 📝 Project Overview
This system helps PSU students track and verify their academic credits efficiently. It is built to streamline the process of graduation checking and academic planning.

### Key Features
* **Student Authentication:** Secure access for PSU students.
* **Credit Tracking:** Real-time calculation of earned vs. required credits.
* **Major-Specific Logic:** Tailored for PSU's curriculum structure.

## 💻 Tech Stack
* **Frontend:** Next.js
* **Styling:** Tailwind CSS
* **Cloud Platform:** Microsoft Azure
